# Metaphors Index — *The Great Gatsby*

A consolidated, cited index of the figurative language catalogued across all nine
chapters. Every entry carries a line-number citation (`l. N`) pointing to the
exact line in the corresponding `plain-text/chapterN/chapterN.md` file, so each
quoted figure can be traced back to its source.

Within each chapter, **Metaphors** lists genuine metaphors; **Similes & related**
lists notable similes and other figurative language (kept separate so the devices
are not conflated). Line numbers refer to the split chapter files, whose first
line is the `## Chapter N` heading.
